# URG-OS Alpha Demo

URG-OS は、グラフネイティブな Code Governance AI です。

このリポジトリは、URG-OS の Alpha Demo 資料を公開するためのものです。
現時点では、コア実装全体は含みません。

URG-OS は、公開 Python OSS リポジトリの静的コード構造を解析し、匿名化された Code Governance Report を生成します。

URG-OS は以下を行いません。

- raw code の保存
- ファイルパスや識別子の保存
- コードの書き換え
- Pull Request の作成
- raw code の LLM 送信

現在は、個人開発者・OSS メンテナからのフィードバックを集める Alpha Demo フェーズです。

## Alpha 無料診断

公開 Python OSS リポジトリを対象に、最大10件まで無料診断を受け付けています。

診断を希望する方は、Issues から「無料リポジトリ診断リクエスト」を作成してください。
